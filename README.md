## If on termux:

◍ Git clone the repo:

    • git clone https://github.com/erenmetesar/NiceGrill

◍ Install the requirements with the lines below:

    • cd NiceGrill
    • pkg install clang
    • pkg install libjpeg-turbo
    • pkg install libwebp
    • pip install cython
    • pip install numpy
    • pip install - r requirements.txt

◍ Run the bot:

    • python - m nicegrill login
